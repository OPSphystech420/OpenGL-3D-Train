### MacOS

#### Requirements
GCC c++17 or later
```
brew install freeglut
brew install glfw3
brew install glew
brew install openal-soft
brew install libsndfile 
```
#### Run
```bash
sed -i '' 's/\r$//' run.sh
chmod +x run.sh
./run.sh
```
